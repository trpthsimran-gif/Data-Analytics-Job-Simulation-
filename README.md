# Data-Analytics-Job-Simulation-

This repository contains my work from the **Quantium Data Analytics Virtual Experience Program (Forage)**.  
In this simulation, I completed real-world style tasks focused on **retail data analysis**, **customer insights**, and **uplift testing** to support business decision-making.

---

## Project Overview
The goal of this simulation was to analyze retail transaction data and provide insights that can help improve business performance.  
I worked on data preparation, customer analytics, experimentation, and final reporting.

---

## Tasks Completed
- **Data Preparation & Cleaning**
  - Cleaned and processed retail transaction datasets.
  - Removed errors, handled missing values, and created structured data for analysis.

- **Customer Analytics**
  - Segmented customers based on purchasing behavior.
  - Identified patterns and trends in customer shopping activity.

- **Uplift Testing / Experiment Analysis**
  - Compared trial vs control stores to measure impact.
  - Evaluated whether changes improved sales performance.

- **Insights & Reporting**
  - Summarized findings into clear insights.
  - Provided business recommendations based on analysis.
